\begin{homeworkProblem}[Q5A]
Assume $I_E = \frac{\beta+1}{\beta}I_C = 1.01I_C \approx I_C$\\

$V_b$ denotes the voltage of the base (reference voltage is GND).\\
$V_b=V_{BE}$+200mV\\

$10I_B(V_{CC}-V_b) + 9I_BV_b + V_{CC}I_C$ = 5mW\\
$I_B = \frac{1}{\beta} I_C = 0.01I_C$\\

Combining three equations yields:\\
$I_C = \frac{5 \times 10^{-3}}{2.748-0.01V_{BE}}$\\

First guess: $V_{BE}$=0.7V\\
$I_C = \frac{5 \times 10^{-3}}{2.748-0.01 \times 0.7} = 1.824mA$\\

$V_{BE}=V_T \ln\frac{I_C}{I_S}$=0.747V\\

Second guess: $V_{BE}$=0.747V\\
$I_C = \frac{5 \times 10^{-3}}{2.748-0.01 \times 0.747} = 1.824mA$\\

$I_C$ converges, hence $V_{BE}$=0.747V\\

$|A_v| = \frac{g_mR_C}{1+g_mR_E} = \frac{\frac{I_C}{V_T}R_C}{1+\frac{I_C}{V_T}R_E} = \frac{I_CR_C}{V_T+I_CR_E} = \frac{I_CR_C}{26mV + 200mV} = 5$\\
$I_CR_C$=1.13V\\
$R_C=\frac{1.13V}{1.824mA}=619\Omega$\\

$R_E=\frac{200mV}{1.824mA}=110\Omega$\\

$V_b=V_{BE}$+200mV=0.947V\\

$R_2 = \frac{V_b}{9I_B} = \frac{V_b}{0.09I_C} = 5769\Omega$\\

$R_1 = \frac{V_{CC}-V_b}{10I_B} = \frac{V_{CC}-V_b}{0.1I_C} = 8514\Omega$\\

Ang LEE
\end{homeworkProblem}
